# Chaves de Identifica o-Digital
Projeto semestral de Novas Tecnologias - Trabalhando desenvolvimento em Python.

# Instalando as bibliotecas

Para rodar o programa precisamos do venv com todas as bibliotecas usadas do flask.

> python -m venv venv
> 
> venv\Scripts\activate
> 
> pip install flask flask-sqlalchemy flask-cors


# Como Rodar?

Após instalar as bibliotecas Flask corretamente, basta iniciar o projeto através do app.py 

>python app.py

Pronto! Servidor rodando e conectado com o banco de dados SQLite!
