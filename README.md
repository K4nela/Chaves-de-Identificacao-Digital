# Olá, Bem vindo! 
Este é um projeto desenvolvido dentro do tema Ciência e Saúde, contemplado o tópico Ferramentas de Educacionais.

# Como rodar o código?
Para roda-lo em sua máquina você precisa:
- clonar o repositório.
- dentro do terminal entrar no diretório "back/" e instalar as bibliotecas flask
> cd back
> python -m venv venv
> venv\Scripts\activate
> pip install flask flask-sqlalchemy flask-cors
//é importante que você esteja usando o terminal cmd
//após a instalação de todas as bibliotecas, vamos iniciar o app.py da seguinte maneira:
>python app.py

- abrir outro terminal para o diretório "front/"
//vamos iniciar o front
> cd ../
> cd front
> npm run dev


Com os dois códigos rodando corretamente, abra a rota que o vite disponibilizou no terminal e navegue pelo sistema.
