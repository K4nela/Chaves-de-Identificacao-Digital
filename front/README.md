# Front end - Chaves de Identifica o-Digital
Projeto semestral de Novas Tecnologias - Trabalhando desenvolvimento em Python.

# Qual framework usamos?
Decidimos usar o React com TypeScript para desenvolver o front que consome a nossa API em Flask.

# Como usar?
Para iniciar o front precisaremos instalar as bibliotecas necessárias.

>npm install

# Como Rodar?
Basta digitar no terminal:

>npm run dev
